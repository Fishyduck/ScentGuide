# Fragrance Lab

A GitHub Pages-ready vanilla HTML/CSS/JS app for organizing fragrances and building formulas from an oil collection.

## Pages
- **Wear Guide**: fragrance cards, filters, temperature matching, daily recommendations.
- **Fragrance Creator**: formula builder with percentage analysis, scent profile, longevity/projection forecast, warnings, and similar-to matching.
- **Oil Library**: classified material database used by the formula engine.

## Run Locally
Open `index.html` directly, or use a local server:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Deploy to GitHub Pages
1. Upload all files in this folder to a GitHub repo.
2. Go to **Settings → Pages**.
3. Set source to your main branch/root.
4. Open the published GitHub Pages URL.

No paid API or AI integration is required.
